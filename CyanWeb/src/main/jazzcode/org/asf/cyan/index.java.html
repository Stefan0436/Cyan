What is Cyan?

Cyan is intended to be a compatible modloader, currently it can run together with Forge, Fabric and even Paper.

Of course, it is still a work-in-progress system.

But you can already go and check it out.


This website serves the pre-build Cyan installers as well as source zips.

The installer selection wizard will provide you with the right installer for the right modloader (and vanilla minecraft)
